Resumen del proyecto Flutter Booking

El proyecto permite que los propietarios se registren y entren con su usuario. 
Los apartamentos se muestran en una lista y se pueden ver con más detalle. Se pueden 
crear, editar y borrar apartamentos fácilmente.

No está disponible la opción de actualizar la lista deslizando hacia abajo.

Es posible subir fotos de los apartamentos desde el móvil y elegir los servicios que ofrece cada uno.
El usuario puede seleccionar imágenes desde la galería y subirlas a Supabase Storage.

Detalle del Apartamento: Existe una pantalla de detalle para cada apartamento, 
donde se muestra información ampliada. 

Se usan efectos visuales como SliverAppBar y Parallax.

En general, el proyecto cumple con lo básico: registro, ver y gestionar apartamentos, subir 
imágenes y seleccionar servicios. La función de refrescar la lista aún no está hecha.
